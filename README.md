# unnecessary-discord-bot
A Discord bot with different types of functionalities

```shell
npm i discord.js @discordjs/rest discord-api-types dotenv
```

- Fill ".env" with:
```json
APP_ID = {APPLICATION_ID}
TOKEN = {YOUR_BOT_TOKEN}
```
